웹 프로그래밍 10주 (1120)

※ 시험문제 : 자바스크립트 객체 생성 방법 (object, 생성자 함수, 객체 리터럴 방식 사용 + 각각 코드 필요)
- 객체의 상태와 행동은 각각 프로퍼티와 메소드로 구현
- 프로그래밍에서 인스턴스란 메모리에 생성된 객체를 의미
- PHP는 클래스의 이름은 대소문자를 구분한다.
- PHP에서 생성자의 이름은 __construct()로 정해져 있다.
- PHP는 소멸자의 이름은 __desturct()로 정해져 있으며, 매개변수를 가질 수 없다.
- PHP는 객체의 이름 뒤에 화살표 기호(->)를 붙이고, 접근하려고 하는 프로퍼티나 호출하고자 하는 메소드의 이름을 사용하면 된다.
- PHP는 클래스 멤버에 public, private, protected 키워드를 사용하여 각각의 멤버에 대한 접근 제어를 명시할 수 있다.
- PHP에서는 단순히 클래스의 정의 내에서 프로퍼티나 메소드를 사용하고 싶을 때는 범위 지정 연산자(::)를 사용할 수 있다.
- PHP에서는 다른 언어와는 달리 프로퍼티나 메소드를 동적으로 '생성한다'는 의미로 오버로딩을 사용한다.
- form 요소에 포함된 input 요소의 name 속성값은 PHP 스크립트에서도 똑같이 사용된다.
- input 정보 검사 할때 tag -> javascript -> PHP 순서로 처리
