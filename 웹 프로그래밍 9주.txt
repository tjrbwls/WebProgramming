웹 프로그래밍 9주 (1113)

- PHP에서 배열은 맵으로 이루어진, 순서가 있는 집합을 의미
- 맵은 한 쌍의 키와 값으로 이루어져 있습니다.
- PHP는 배열 초기화에 괄호를 사용한다. (일반적인것은 중괄호 사용)
- count() 함수는 배열 요소의 개수를 계산할 때 배열의 홀을 포함하지 않는다.
- 인덱스에 대응하는 배열 요소가 없는 부분을 배열의 홀이라고 한다.
- PHP는 2차원 배열은 1차원 배열과 같이 array() 함수를 사용하여 선언하며 1차원 배열과 다른 점은 array() 함수를 한 번 더 사용하여, 배열 요소에 또 다른 1차원 배열을 생성한다는 점이다.
- PHP는 키와 값 사이에 '=>'를 넣어 주어 배열 요소에 저장될 키와 값을 지정할 수 있습니다.
- PHP는 foreach 문을 사용하면 연관 배열 요소의 키값과 요소의 값을 변수에 따로 저장하여 루프 내에서 사용할 수 있습니다.
- PHP에서 함수를 작성 할때는 함수의 이름은 문자와 숫자, 언더스코어만 사용할 수 있다.
- PHP는 함수 오버로딩을 지원하지 않으므로, 이미 선언된 함수를 다시 선언할 수 없다.
- passing by value : 매개변수에 저장된 값은 전달받은 데이터의 복사본으로, 함수 안에서 변경되어도 함수 밖의 원본 데이터에는 영향을 주지 않는다.
- passing by reference : 함수 내부에서 함수 밖의 데이터를 조작하기 위해서는 해당 변수를 전역 변수로 선언할 수도 있다. (사용시 '&'사용)
- 가변 길이 인수 목록 : '...' 토큰을 사용하여 간편하게 설정할 수 있다.
- PHP에서 모든 함수는 전역 함수이므로, 함수 내부에서 선언된 함수라도 해당 함수 외부에서 호출할 수 있다.
- 가변 함수 : 변수를 사용하여 함수를 호출하는 것
- PHP 4.2.0부터는 "boolean" 대신에 "bool"을 사용할 수 있으며, "integer" 대신에 "int"를 대신 사용할 수 있다.
- UTF-8 인코딩 방식에서는 영문자는 한 문자당 1바이트, 한글은 한 문자당 3바이트로 표현된다.
- 모든언어는 정규 표현식을 지원한다.
